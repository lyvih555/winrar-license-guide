## WinRAR 是什么？

WinRAR 是由德国 RARLAB 公司开发的老牌压缩软件，1995 年上市，到现在快 30 年了。核心卖点就俩字：**稳**和**全**。

- **格式支持广**：创建 RAR/ZIP 压缩包，解压 7Z、CAB、JAR、ISO 等十多种格式
- **分卷压缩**：大文件分段切割，方便传输和存储，传统网盘用户的福音
- **自解压文件**：生成一个 `.exe`，对方不安装任何软件，双击就能解压
- **AES 加密 + 密码管理器**：给压缩包加上密码，还能自动记住、自动匹配，敏感文件不怕泄露
- **内置安全扫描**：解压前检测压缩包是否含有潜在威胁，减少踩雷风险
- **修复损坏压缩包**：别的软件遇到损坏压缩包直接报错，WinRAR 有时候能救回来
